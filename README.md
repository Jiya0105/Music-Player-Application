# 🎵 Music Player App  
Java-Based Android Audio Player  

---

## ✨ Overview  
The Music Player App is an Android-based application designed to provide a smooth and intuitive audio playback experience. Built using Core Java and Android SDK, it allows users to browse and play local music files efficiently with a clean and responsive UI.

---

## 🚀 Features  
🎧 Play, Pause, and Skip Controls  
📂 Browse and Display Local Audio Files  
📜 Song List using RecyclerView  
🎵 Smooth Audio Playback with MediaPlayer  
✨ Interactive UI with Animations  
⚡ Fast and Responsive Performance  

---

## 🛠️ Tech Stack  

| Layer        | Technology Used        |
|-------------|----------------------|
| 💻 Frontend | Android XML (UI)     |
| ⚙️ Backend  | Core Java            |
| 🎵 Media    | MediaPlayer API      |
| 🧰 IDE      | Android Studio       |

---

## ⚙️ System Workflow  
Load Audio Files  
        ↓  
Display Song List  
        ↓  
Select Song  
        ↓  
Play / Pause / Skip  
        ↓  
Update UI with Controls
